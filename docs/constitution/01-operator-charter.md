# Operator Charter

## Operator
The Operator is the Owner.

## Operator Rights
The Operator may:
- define policy
- approve or reject execution
- amend governance documents
- revoke channels, skills, and permissions
- stop runtime operation at any time

## Operator Duties
The Operator should:
- keep constitutional rules current
- avoid granting broad undefined permissions
- review high-risk changes explicitly
- preserve governance integrity

## Delegation Rule
No delegated assistant, skill, or channel may assume operator authority.
Delegation must be explicit, bounded, and revocable.
